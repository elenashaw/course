# 国际单位制（SI）词头

**往上走 ($10^3$)**：Kilo (K) $\to$ **Mega (M)** $\to$ Giga (G) $\to$ Tera (T) $\to$ **Peta (P)** $\to$ Exa (E) $\to$ **Zetta (Z)** $\to$​ **Yotta (Y)**

* KiMi GoT a PEt ZoYe

| **单词** | **对应词头**               | **指数**            | **巧妙记忆点**                                 |
| -------- | -------------------------- | ------------------- | ---------------------------------------------- |
| **Ki**   | **Ki**lo (K)               | $10^3$              | **千**。基础单位。                             |
| **Mi**   | **Me**ga (M)               | $10^6$              | **百万**。Millionaire（百万富翁）。            |
| **Go**   | **Gi**ga (G)               | $10^9$              | **十亿**。Billion（注意：Giga 对应的是十亿）。 |
| **T**    | **Te**ra (T)               | $10^{12}$           | **万亿**。Trillion。                           |
| **PEt**  | **P**eta (P) + **E**xa (E) | $10^{15}$,$10^{18}$ | **拍/艾**。P 后面紧跟 E，逻辑丝滑。            |
| **Zo**   | **Ze**tta (Z)              | $10^{21}$           | **泽**。Z 是倒数第二个字母。                   |
| **Ye**   | **Yo**tta (Y)              | $10^{24}$           | **尧**。Y 是字母表末尾，代表极限。             |

* 一般都是a结尾



**往下走 ($10^{-3}$)**：milli (m) $\to$ micro ($\mu$) $\to$ nano (n) $\to$ pico (p) $\to$ femto (f) $\to$ atto (a) $\to$ zepto (z) $\to$ yocto (y)

* "Meow(**m**， **$\mu$**), **n**o **p**etting my **f**ur unless you give me an **a**pple." --**Z**o**y**e 

| **单词**  | **对应词头**      | **指数**   | **实际含义 (逻辑关联)**                       |
| --------- | ----------------- | ---------- | --------------------------------------------- |
| **mi**    | **mi**lli (m)     | $10^{-3}$  | **毫**。如毫米（millimeter），雨滴级别。      |
| **$\mu$** | **micro** ($\mu$) | $10^{-6}$  | **微**。如微秒、微米（micron），细胞级别。    |
| **n**     | **n**ano (n)      | $10^{-9}$  | **纳**。如纳米（nanometer），分子级别。       |
| **p**     | **p**ico (p)      | $10^{-12}$ | **皮**。如皮秒，原子级别。                    |
| **f**     | **f**emto (f)     | $10^{-15}$ | **飞**。如飞秒（femtosecond），激光脉冲级别。 |
| **a**     | **at**to (a)      | $10^{-18}$ | **阿**。如阿托秒，电子运动级别。              |
| **z**     | **ze**pto (z)     | $10^{-21}$ | **仄**。极微小的亚原子尺度。                  |
| **y**     | **yo**cto (y)     | $10^{-24}$ | **幺**。目前定义的最小词头。                  |

* 一般都是o结尾
